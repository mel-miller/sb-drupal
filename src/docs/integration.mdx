# Drupal Integration

This toolkit is intended to be imported directly into a custom Drupal theme.

It is recommended that you build your theme from scratch and set the base theme to `stable9`.

## Prerequisites

- Drupal 10 or 11 site with a custom theme.
- Drupal [Components!](https://www.drupal.org/project/components) module installed and enabled.
- Node.js and NPM installed on your local machine.
